# **Code Document Header Template**

This template is designed to be placed at the very top of code files (e.g., .jsx, .html, .js) for quick audit and version tracking.

\# Module: MOD\_XXX\_000  
\# EU WP Code: WPX\_DXX.X  
\# Version: v01  
\# Conceptual Author: \[You\]  
\# Documenting Author: \[Gemini LLM\]  
\# Created: \[YYYY-MM-DD\]  
\# Git Commit: \[a1b2c3d\]  
\# SHA256: \[insert checksum after freeze\]  
\# Status: Draft  
\# ─────────────────────────────────────────────
